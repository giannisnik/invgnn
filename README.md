# InvGNN: Learning Invertible Node Representations on Graphs

This repository is the official implementation of [InvGNN: Learning Invertible Node Representations on Graphs](https://openreview.net/pdf?id=okKDijCwRn).

## Requirements

Code is written in Python 3.13 and requires:
* PyTorch 2.7.1
* PyTorch Geometric 2.6.1
* NetworkX 3.4.2
* scikit-learn 1.6.1


## Datasets

The datasets are automatically downloaded when the code is executed
